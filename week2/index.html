Week 2 Notes

JavaScript: Novice to Ninja, 2nd Edition

Ch2: Programming basics

Primitive data types:

  • String
  • Symbol
  • Number
  • Boolean
  • Undefined
  • Null

Variables:

  • let: This variable can be reassigned
  • const: Can not be changed/reassigned

Global Scope: accessible everywhere in the program (Not good practice)
Local Scope: variables designed in a block are only accessible in that block.

A method is an action that an data type or an object can perform.
example: name.toUpperCase();
This makes all of the letters in this variable uppercase.

Template literals: A special kind of String.
Use the backtick character, ` , This acts as both kinds of quotation marks.
These strings can have expresions inserted inside of them. This way you can concat a string and a variable.

Ch3: Arrays, Logic, and Loops.

Arrays are ordered lists of values.

  • Pop() : remove the last item from an array
  • Push() : appends a new value to the end of the array
  • Shift() : removes thefirstitem in the array
  • Unshift() : appends a new item to thebeginningof the array

Logical conditions allow you to control how a block of code runs.

Switch statements allow you to test muliple conditions.

Nested for loops: A loop inside another loop.

Ch4: Functions.

Invoking a function is just running the code inside the function.
You need paratheses() after a variable in order to invoke a function.

All functions have a name. The name property helps us access the function.

Function Declaration example:
function hello(){ console.log('Hello World')};

Function expression: assigns anonymous function to a variable. example:
const goodbye = function(){ console.log('Goodbye World!'); };

Function Constructors: Avoid these, they are not good practice.

Arrow Function: A new syntax that uses an 'arrow' symbol =>
example: const square = x => x*x;

Array Iterators: forEach() map () reduce() filter()
Array Iterators can be chained together.

JavaScript: Novice to Ninja, 2nd Edition

Chapter 5: Objects

  • Objects are found inside curly braces which is holding key values
  • JSON is another way to store data
  • JSON uses object literals

Chapter 6: Document Object Model

  • Nodes are objects that can modify or locate informataion in the body element
  • Nodes are like arrays but they are NOT actually arrays
    • Node types and their codes: 1 element, 2 attribute, 3 text, 8 comment, 9 body
  • DOM Shortcuts (direct text from chapter):
    • Document.body : returns the body element of a web page
    • Document.images : returns a node list of all the images contained in the document
    • Document.links : returns a node list of all theelements andelements that have anhrefattribute.
    • Document.anchorsreturns : a node list of all theelements that have anameattribute.
    • Document.formsreturns a node list of all the forms in the document.
  • It's possible to get elements by their Classes and ID's
  • Nodes can have children that can be individually selected

Chapter 7: Events

  • Event listeners load the page when they are clicked
  • Event objects can be triggered in different ways. Example: click, mouse down, mouse up, and different keys on the keyboard
  • Event listeners can be removed
  • Event propagation: elements can be nested inside one element and be activated by just one click
  • There is no such thing as swipe events

This is JavaScript

  • Method is a fancy word for a function
  • The keyword "this" basically points to objects
  • "this" triggers events when they are set to elements

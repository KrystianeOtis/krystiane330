Chapter 12: Object Oriented Programming

  • JavaScript uses Object Oriented Programming alot like Python does
  • Encapsulation, Polymorphism, Inheritance, and Classes are all used in object oriented programming
  • Chaining functions is done to form a sequence of functions/methods
  • Can use "that" to equal "this" to set a variable before a nested function
  • Can use Mixins instead of inheritance to add properties to a method

Chapter 15: Modern JavaScript

Section on Modular JavaScript

  • Modules have their own global scope
  • A Module is a self-contained chunk of code - They provide functions/methods that can be used in other files
  • Functions can be exported from a module
  • HTML-style comments CANNOT be used in modules
  • Node.js already implemented modules before it was cool
